# Developed an end-to-end Loan Default Prediction system with class imbalance handling (SMOTE), hyperparameter tuning, and deployed using FastAPI as a REST API.

End-to-end Machine Learning project that predicts loan default risk using classification models and deploys the model using FastAPI.

## Tech Stack

->Python
->Pandas
->Scikit-learn
->SMOTE
->FastAPI
->Uvicorn
->MLflow
->Joblib

## Features

->EDA
->Feature Engineering
->Class Imbalance Handling (SMOTE)
->Cross Validation
->Hyperparameter Tuning
->Model Tracking (MLflow)
->REST API Deployment
